Document Structure

Every HTML page starts with a document declaration: <!DOCTYPE html>, which tells the browser this is an HTML5 document. :contentReference[oaicite:1]{index=1}

The root element of an HTML document is: <html>. All content appears inside this tag. :contentReference[oaicite:2]{index=2}

The <head> section contains metadata such as the page title: <title>, links to stylesheets, and character encoding.
