Document Structure

<!DOCTYPE html> declares the HTML5 document type. The <html> tag is the root of the HTML document. Everything shown in the browser goes inside <body> while metadata like titles and styles go in <head>.

<!DOCTYPE html>
<html>
 <head>…</head>
 <body>…</body>
</html>

Text and Content Tags

Headings (<h1>–<h6>) define titles of different importance. Paragraphs (<p>) show blocks of text, and lists (<ul>, <ol>) organize items.

<h1> to <h6>

Used for main headings and subtitles.

<p>

Shows a paragraph of text.

<ul> & <ol>

Unordered and ordered lists.

Links and Media

The <a> tag creates clickable links. Images (<img>) embed graphics into your web pages.

<a href="https://www.w3schools.com/html">Visit HTML Tutorial</a>
<img src="https://via.placeholder.com/150" alt="Example Image">
